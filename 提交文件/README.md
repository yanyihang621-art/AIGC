# 寻迹华夏 · 提交作品准备

本目录用于收录提交作品所需的相关信息与说明文件。

## 1. 项目概览

“寻迹华夏”是一个基于 **新中式美学** 设计的交互式 AI 历史文化地图系统。用户可以通过点击中国省级地图，探索该省份在不同历史时期（从远方至先秦到明清）的文化叙事，并通过微信式的“会名士”对话系统，与相应的历史人物进行跨时空的流式角色扮演对话。

### 核心功能板块
1. **AI 动态文化地图**：集成 SVG 中国地图，支持滚轮平滑缩放与拖拽。
2. **时空文化探寻**：动态加载对应省份和朝代的文化介绍，支持流式打字渲染与客户端/服务端双重缓存。
3. **会名士 (WeChat-style NPC Chat)**：
   - **名士名录侧边栏**：左侧提供传统竹简风格的名士列表，支持在不同名士间平滑切换。
   - **聊天记忆系统**：支持独立存储每位名士的对话历史，切换聊天不丢失记录。
   - **极致身份锁定**：后端精准解析所选名士，设定严格的身份锁定提示词与低温参数（0.6），杜绝身份漂移或AI出戏。
   - **古书风格排版**：不使用气泡，采用古籍纸张、红色客/士印章与手写毛笔光标。

## 2. 目录与文件清单

提交时建议附带以下文件，以证明项目的完整性与工程水准：

* `/dist`：生产环境编译打包后的前端静态资源（HTML/CSS/JS）。
* `/src`：前端核心源码，包括交互逻辑与样式系统。
* `/api`：Serverless 云函数服务部署代码（适用于 Vercel/Netlify 等无服务器环境）。
* `server.js`：本地 Express 中转代理服务，支持本地极速启动与 SSE 流式传输。
* `.env`：环境变量配置文件模板（需填入 `DEEPSEEK_API_KEY`）。
* `PRODUCT.md` & `DESIGN.md`：产品规格书与设计指南。

## 3. 本地启动与部署步骤

### 运行环境要求
* Node.js v18.0 或更高版本
* 网络环境通畅（需调用 DeepSeek 接口）

### 本地极速启动
1. **安装依赖**：
   ```bash
   npm install
   ```
2. **配置 API Key**：
   在根目录创建 `.env` 文件，并写入你的 DeepSeek 密钥：
   ```env
   DEEPSEEK_API_KEY=your_actual_api_key_here
   PORT=3001
   ```
3. **启动后端服务**（支持 SSE 流式中转与多重缓存）：
   ```bash
   node server.js
   ```
4. **启动前端开发服务器**：
   ```bash
   npm run dev
   ```
5. 打开浏览器访问控制台输出的前端地址（默认 `http://localhost:5173/`）。
